# pokemon_128px_by_lancelotxiii_dj8sk6x

Pack numerote de sprites `128PX`, attribue a LancelotXIII d'apres le nom du
dossier.

## Contenu

- Fichiers PNG nommes sur le schema `0006-WUX_128PX.png`.
- Le prefixe numerique semble correspondre a un identifiant de Pokemon.
- Le suffixe `128PX` indique une taille cible ou un export en 128 pixels.

## Usage possible

- Sprite local alternatif pour certaines especes.
- Icones plus grandes pour fiches, vitrines ou combats.
- Pack source a reclasser sous `pokemon_sprite/` si son usage est confirme comme
  strictement Pokemon.

## Attention

Ce pack est actuellement dans `trainer_sprite/` alors que son nom indique
`pokemon_128px`. Ne pas le deplacer sans verifier les usages prevus et les futurs
mappings.
