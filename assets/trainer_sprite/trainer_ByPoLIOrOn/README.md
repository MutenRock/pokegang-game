# trainer_ByPoLIOrOn

Pack de sprites de dresseurs attribue a PoLIOrOn d'apres le nom du dossier.

## Contenu

- `leader_of_the_elite_4_by_polloron_.png`, image source ou reference.
- Variantes `trainer_PoLlOrOn_*`, parfois avec suffixes `A`, `B` ou `C`.

## Usage possible

- Dresseurs rares ou elites.
- Gym leaders, Elite 4 ou adversaires d'evenement.
- Variantes de boss ou skins cosmetiques.

## Integration

- Garder les variantes `A`, `B`, `C` distinctes dans le mapping.
- Tester la lisibilite sur fonds sombres.
- Documenter toute association a un personnage nomme.
