# trainer_ByKyledove

Pack de sprites de dresseurs attribue a Kyle Dove d'apres le nom du dossier.

## Contenu

- `trainer_battle_sprites_by_kyle_dove_dboanpx.png`, probablement la planche
  source ou image de reference.
- `trainer_kyledove_01_A.png` a `trainer_kyledove_59_A.png`.

## Usage possible

- Galerie de choix du boss.
- Dresseurs generiques ou adversaires de zone.
- Variantes de PNJ pour missions et evenements.

## Integration

- Garder les numeros existants comme identifiants source.
- Creer des alias lisibles uniquement dans les mappings de donnees.
- Verifier le cadrage en carte agent, combat et modale de selection.
