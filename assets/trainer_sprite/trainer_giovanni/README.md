## Usage possible

- Boss de reference dans l'intro.
- Skin ou hommage pour le boss joueur.
- Adversaire special en combat narratif.

## Integration

- Preferer des alias stables dans le code, par exemple `giovanni_hgss`.
- Eviter d'utiliser directement les noms avec `!` dans les chemins runtime si un
  mapping plus propre est possible.
- Verifier le format exact avant affichage dans les composants existants.
