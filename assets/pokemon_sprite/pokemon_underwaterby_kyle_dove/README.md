# pokemon_underwaterby_kyle_dove

Pack de sprites underwater attribue a Kyle Dove d'apres le nom du dossier.

## Contenu

- `sprite_underwater_001.png` a `sprite_underwater_112.png`
- `underwater_overworlds__d50geej.png`, probablement la planche source ou image
  de reference du pack

## Usage possible

- Variantes overworld pour zones aquatiques.
- Sprites d'ambiance pour Safari, routes maritimes ou evenements speciaux.
- Base de mapping pour des especes aquatiques si le jeu ajoute un mode sprite
  local dedie.

## Integration

- Conserver l'ordre numerote tant que le mapping espece par espece n'est pas
  documente.
- Ajouter un fichier de mapping explicite si ces sprites deviennent utilises par
  le runtime.
- Verifier la transparence et la lisibilite sur fonds sombres et clairs.
