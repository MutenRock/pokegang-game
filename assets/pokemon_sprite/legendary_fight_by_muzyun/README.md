# legendary_fight_by_muzyun

Pack de sprites/compositions de combat legendaires attribue a Muzyun d'apres le
nom du dossier.

## Contenu

| Fichier | Note |
|---|---|
| `darkrai_fight_full.png` | Composition Darkrai pleine. |
| `darkray_fight_half.png` | Variante Darkrai/Darkray recadree. Nom a conserver tant que les mappings ne sont pas verifies. |
| `darkray.png` | Sprite isole Darkrai/Darkray. Nom a conserver tant que les mappings ne sont pas verifies. |
| `deoxys_fight_full.png` | Composition Deoxys pleine. |
| `deoxys_fight_half.png` | Variante Deoxys recadree. |
| `deoxys.png` | Sprite isole Deoxys. |
| `mewtwo_fight_full.png` | Composition Mewtwo pleine. |
| `mewtwo_fight_half.png` | Variante Mewtwo recadree. |
| `mewtwo.png` | Sprite isole Mewtwo. |
| `legendary_pokemons_wants_to_fight__by_muzyun.png` | Image source ou planche de reference du pack. |

## Usage possible

- Boss fights legendaires.
- Cinematiques de raid ou d'evenement.
- Variantes speciales dans les modales de combat.

## Attention

Le pack contient une incoherence de nommage `darkrai` / `darkray`. Ne pas
renommer sans verifier les futurs mappings ou references.
